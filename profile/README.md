# Q.E.D. Git & GitHub Study

- Git과 GitHub의 개념
- 기존 협업 과정에서의 문제

- 원격 저장소 업로드 및 최신 정보 반영

### Git & GitHub 2
- 브랜치 생성, 전환, 삭제
- 브랜치 병합
- Merge Conflict
- 커밋 수정 및 복구
- 특정 커밋 선택 적용
- `.gitignore` 파일 설정
- 이슈 및 협업 기능
- Special Repository
- GitHub Desktop 활용
